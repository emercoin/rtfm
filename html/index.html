Emercoin (EMC) - это криптовалюта с гибридной PoS и PoW добычей (при этом объединенная с биткойнами). Генерация монет ориентирована в основном на производство монет с помощью энергосберегающих PoS, поскольку сложность PoW постепенно увеличивается с течением времени.

Общий объем предложения

Алгоритмическое увеличение прибл. 6% в год, статистика.

Частота создания блока

В среднем за 10 минут

Алгоритм PoW

SHA-256 (объединенный с биткойнами).

Вознаграждение за блок PoW

5020 EMC, уменьшается в зависимости от сложности PoW.

Формула

Награда = 5020 / sqrt (sqrt (сложность))

Награда PoS

Приблизительно 6% годовых.

Срок созревания монет для PoS

30 дней.

Максимальный возраст монет для PoS

90 дней.

Регулировка сложности

Пересчет идет каждый блок. Зависит от сложности PoW

Срок зрелости вновь добытой награды

32 блока, пока не будут потрачены новые монеты.

Base58Check кодировка префикс адреса (en)

33

Порт p2p по умолчанию

6661

Порт RPC по умолчанию

6662
